## ch09. [Express][express] 프레임워크.

* 프로젝트 생성.
`express HelloExpress`
* 프로젝트 디펜던시 설정.
`cd HelloExpress && npm install`
* 서버 실행
`node app`

* 디펜던시 설정.
`package.json`

```json
{
  "name": "application-name",
  "version": "0.0.1",
  "private": true,
  "scripts": {
    "start": "node app.js"
  },
  "dependencies": {
    "express": "3.2.6",
    "jade": "*"
  }
}
```

express -ssession --template ejs HelloExpress



* production모드로 설정.
 - NODE_ENV=production node app
 - app.settings.env
